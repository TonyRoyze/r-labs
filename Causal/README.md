# Curl Snippets

This folder contains small, single-purpose markdown snippets you can upload to GitHub and fetch individually with `curl`.

Each file is designed for a recurring causal inference question type rather than for one worksheet number.

Raw GitHub pattern:

```bash
curl -L https://raw.githubusercontent.com/<user>/<repo>/<branch>/curl_snippets/<file>.md
```

Start with [WHEN_TO_CURL.md](/Users/vidura/Code/CausalML/curl_snippets/WHEN_TO_CURL.md) to choose the right snippet.

For Windows clipboard usage, see [WINDOWS_CLIPBOARD.md](/Users/vidura/Code/CausalML/curl_snippets/WINDOWS_CLIPBOARD.md).
